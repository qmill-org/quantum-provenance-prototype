# Conceptual API example (illustrative)

This directory contains a conceptual provenance-response example using conceptual
schema version `1.0.0`.

- The example combines provider-supplied, derived, and application-captured
  fields used in the paper's broader conceptual API discussion.
- It is not a provider response.
- It is not generated by the evaluated `0.1.0-prototype` contract.
- It is separate from schema-valid records generated under
  [`artifact/examples/`](../../artifact/examples/) and from the generated
  supplementary Braket example at
  [`../complete-example-provenance-response.json`](../complete-example-provenance-response.json).
